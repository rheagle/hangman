# Hangman


## To play this game:
### 1. Open Terminal / Command Prompt
### 2. Install Ruby
  **Mac:**
  ```bash
  brew install ruby
  ```
  **Windows:**
  ```bash
  ruby -v
  ```
### 3. Install Bundler and check that it is working
  ```bash
  gem install bundler
  ```
  ```bash
  bundler -v
  ```
### 4. Clone this repo
  ```bash
  git clone https://github.com/your-username/your-repo.git
  cd your-repo
  ```
### 5. Install dependencies in the Gemfile.
  ```bash
  bundle install
  ```
### 6. Run the game!
  ```bash
  ruby main.rb
  ```
